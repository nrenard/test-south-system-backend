# Instalação

Requisitos para rodar o projeto nodeJS >= 8. Clonar repositório, entrar na pasta e instalar suas dependências com o comando `yarn` ou `npm install`.

## Para rodar o server

```
yarn start
```

## Para rodar os testes

```
yarn test
```

## Para verificar a cobertura de testes

```
yarn coverage
```
